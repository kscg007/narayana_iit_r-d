:to remove anything from versioning:
git rm -r --cached myen/
git commit -m "<Comment>"
git push origin <branch-name>


:create new branch locally:
git checkout -b <branch-name/feature-name>


:create new branch in github:
git push -u origin <branch-name/feature-name>




:to push something to a branch in github:
git add .
or
git add <specific names or locations>

git add status

git commit -m "<Comment>"

git push origin <branch-name>


then check and create pull request and merge with master in github.


:after successfull pull request to master in github:

git checkout master # to select master branch locally
git pull origin master # to pull the lastest from the github master
git branch -d <branch-name> # to delete the local sub-branch (best practice to delete local sub-branch after successfully merging with github master)
git branch -D <branch-name> # to delete forcefully without warning (better to run -D command after -d command)

then create new branches locally and github for new feature.


:to delete github branch:
git push origin --delete <branch-name>
